# Personal Website

Ultra-minimal personal website. Pure HTML and CSS. No frameworks, no build tools, no JavaScript.

## Features

- Monospace typography
- Black and white
- Fast loading
- Mobile responsive
- ~145 lines of CSS

## Setup

1. Edit `index.html`:
   - Update your name
   - Update description
   - Add your blog posts
   - Update contact links

2. Customize `style.css` if needed:
   - Change font: Line 8
   - Change colors: Lines 11-12
   - Change max width: Line 13

3. Push to GitHub and enable Pages

## File Structure

```
.
├── index.html    # Main page
├── style.css     # Minimal styling
└── README.md     # This file
```

## Adding Blog Posts

Copy this structure in the `#writing` section:

```html
<article>
    <h3>Post Title</h3>
    <time>2025-11-20</time>
    <p>Post content...</p>
</article>
```

## Design Philosophy

- Minimal: No unnecessary elements
- Fast: Plain HTML/CSS loads instantly
- Readable: Monospace font, good spacing
- Accessible: Semantic HTML, high contrast
- Maintainable: Simple code, easy to edit
